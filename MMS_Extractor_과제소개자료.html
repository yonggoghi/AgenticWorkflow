MMS Extractor - 지능형 광고 메시지 분석 시스템

과제 소개 및 설명 자료

1. 과제 개요

🎯 프로젝트 목적

  • MMS 광고 메시지 자동 분석 시스템 개발
  • 한국어 마케팅 메시지에서 구조화된 정보 자동 추출
  • AI/ML 기술을 활용한 지능형 텍스트 분석 솔루션

📊 주요 성과 지표

  • 94,279개 상품 정보 데이터베이스 구축
  • 5개 주요 LLM 모델 지원 (Gemma, GPT, Claude, AX, Gemini)
  • 3가지 추출 모드 (NLP, LLM, RAG) 제공
  • 25초 → 즉시 응답 성능 개선 달성

🏆 핵심 가치

  • 마케팅 메시지 분석 자동화로 업무 효율성 대폭 향상
  • 다양한 LLM 모델 지원으로 유연한 분석 옵션 제공
  • REST API 제공으로 시스템 통합 용이성 확보

2. 기술적 성과

🚀 핵심 기술 혁신

모델 사전 로딩 시스템

  • 기존: 첫 요청 시 25초 이상 소요
  • 개선: 서버 시작 시 모델 사전 로딩으로 즉시 응답
  • 효과: 사용자 대기시간 95% 단축

하이브리드 추출 방식

  • NLP 모드: 규칙 기반 빠른 처리 (~10-15초)
  • LLM 모드: 지능형 유연한 분석 (~12-25초)
  • RAG 모드: 지식 기반 정확도 향상 (~15-20초)

병렬 처리 최적화

  • 단일 메시지: 병렬 처리로 2.5배 성능 향상
  • 배치 처리: 4워커 사용 시 4배 처리량 증대
  • 메모리 효율성: 인스턴스 캐싱으로 리소스 최적화

🔬 기술적 차별점

  • 다중 LLM 지원: 용도별 최적 모델 선택 가능
  • 유연한 데이터 소스: 로컬 CSV + Oracle DB 동시 지원
  • 실시간 시각화: DAG 그래프로 엔티티 관계 표현

4. 핵심 기능

📝 정보 추출 기능

  • 대리점: 오프라인 매장 정보

프로그램 분류

  • 타겟 고객: 연령대, 관심사별 세분화

지능형 매칭

6. 다양한 활용 방식

🖥️ CLI (Command Line Interface)

python mms_extractor.py --message "광고 메시지" --llm-model ax
  • 개발/테스트: 빠른 단일 메시지 분석
  • 디버깅: 상세 로그 및 단계별 확인
  • 프로토타이핑: 새로운 모델/설정 실험

🌐 REST API 서버

POST /extract
{
  "message": "광고 메시지",
  "model": "gemma_3",
  "extraction_mode": "nlp"
}
  • 시스템 통합: 기존 서비스와 쉬운 연동
  • 확장성: 다중 클라이언트 동시 지원
  • 모니터링: 실시간 상태 확인 및 로깅

📦 배치 처리 시스템

python batch.py --batch-size 100 --llm-model gpt
  • 대량 처리: 수천 건 메시지 자동 분석
  • 스케줄링: 정기적 배치 작업 실행
  • 결과 관리: CSV 파일 자동 저장 및 상태 업데이트

🎨 웹 데모 인터페이스

  • 직관적 UI: 드래그 앤 드롭 방식 메시지 입력
  • 실시간 결과: 즉시 분석 결과 확인
  • 시각화: DAG 그래프 실시간 표시

7. 기술 스택 및 모델

🤖 지원 LLM 모델

프로덕션 모델

  • Gemma 3: Google 개발, 한국어 최적화 (기본 권장)
  • AX: 고성능 추론, 빠른 응답 속도
  • Claude 3.7: Anthropic, 높은 추론 능력

고급 모델

  • GPT-4.1: OpenAI, 범용 높은 성능
  • Claude Sonnet 4: 최신 버전, 고급 분석
  • Gemini: Google, 멀티모달 지원

🛠️ 기술 스택

핵심 기술

  • Python 3.8+: 메인 개발 언어
  • LangChain: LLM 통합 프레임워크
  • Sentence-Transformers: 임베딩 모델
  • Kiwi: 한국어 형태소 분석

데이터 및 저장

  • Pandas/NumPy: 데이터 처리
  • Oracle/CSV: 유연한 데이터 소스
  • JSON: 표준 데이터 교환 형식

웹 및 API

  • Flask: REST API 서버
  • CORS: 크로스 오리진 지원
  • HTML5/CSS3: 웹 인터페이스

🎯 모델 선택 가이드

용도 추천 모델 특징
일반 운영 Gemma 3 균형잡힌 성능, 빠른 속도
고정밀 분석 GPT-4.1 최고 정확도, 복잡한 텍스트
대량 처리 AX 빠른 처리, 비용 효율적
실험/개발 Claude 안정적 성능, 좋은 설명력
